Tomato Game

A project based on the API provided by the University Of Bedfordshire. This project has been created with flutter.

What this project is about

This application was created as part of an assignment to show the 4 major software engineering principles:

1.) Low Coupling High Cohesion

2.) Interoperability

3.) Event Driven Architecture and programming

4.) Virtual Identity
